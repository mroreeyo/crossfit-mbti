# 📋 PRD (Product Requirements Document)
## 🏋️‍♂️ "나의 크로스핏 MBTI는?" — 바이럴 퀴즈 웹사이트

---

## 1. 프로젝트 개요

### 1.1 제품명
**"나의 크로스핏 MBTI는?"** (CrossFit MBTI Quiz)

### 1.2 목적
MBTI 밈 트렌드를 크로스핏에 접목한 인터랙티브 퀴즈 웹사이트를 통해 바이럴 마케팅을 실현하고, 크로스핏 커뮤니티 유입 및 브랜드 인지도를 높인다.

MBTI 테스트는 단순한 성격검사를 넘어 기업을 알리거나 구매 전환을 일으키는 마케팅 수단으로 많이 활용되고 있으며, 실제로 광고 마케팅 없이 18만 뷰를 달성한 사례도 존재합니다 [2].

### 1.3 핵심 성공 지표 (KPI)
| 지표 | 목표 |
|------|------|
| 퀴즈 완료율 | 70% 이상 |
| 결과 공유율 | 40% 이상 |
| 월간 방문자 수 (MAU) | 런칭 1개월 내 50,000 |
| 평균 체류 시간 | 2분 이상 |
| SNS 공유 수 | 10,000+ |

---

## 2. 타겟 유저

### 2.1 Primary Persona
- **연령**: 20~35세
- **특성**: 크로스핏을 현재 하고 있거나 관심 있는 MZ세대
- **행동**: MBTI 관련 콘텐츠를 즐겨 공유하며, 인스타그램/카카오톡 활발 사용

### 2.2 Secondary Persona
- **연령**: 18~40세
- **특성**: MBTI에 관심 많은 일반인 (크로스핏 비경험자)
- **행동**: 재밌는 테스트 결과를 SNS에 공유하는 습관

MBTI 테스트 제작 시 가장 중요한 포인트는 **① 타겟의 관심사 반영, ② 홍보하고자 하는 기업/단체와의 연관성, ③ 참여를 불러일으키는 디자인 컨셉** 세 가지입니다 [2].

#### 3.1.2 퀴즈 문항 (12문항)
**형식**: 각 문항당 2개 선택지 (A/B), 카드 스와이프 or 버튼 클릭

**문항 예시**:

| # | 질문 | A (선택지) | B (선택지) | 측정 축 |
|---|------|-----------|-----------|---------|
| 1 | WOD 시작 전, 나는? | 옆 사람이랑 수다 떨며 워밍업 | 혼자 조용히 모빌리티 루틴 | E / I |
| 2 | 새로운 동작을 배울 때? | 일단 해보고 몸으로 익힘 | 코치 시범을 꼼꼼히 관찰 후 시작 | S / N |
| 3 | 팀 WOD에서 팀원이 힘들어할 때? | "파이팅!!" 크게 외쳐줌 | 조용히 옆에서 페이스 맞춰줌 | T / F |
| 4 | WOD 프로그래밍은? | 매일 다른 게 좋아, 변화가 재밌지 | 정해진 프로그램을 꾸준히 따르는 게 좋아 | J / P |
| 5 | PR 갱신했을 때 나는? | 박스 전체에 알림 (소리 지름) | 조용히 노트에 기록하고 속으로 뿌듯 | E / I |
| 6 | 오늘의 WOD가 "머프(Murph)"라면? | "오 드디어! 도전이다!" | "왜 하필 오늘…" 멘탈 정리 시작 | S / N |
| 7 | 크로스핏 장비를 살 때? | 성능/리뷰 꼼꼼히 비교 분석 | 디자인/색상이 마음에 들면 바로 구매 | T / F |
| 8 | 박스에 늦을 것 같을 때? | 그래도 가야지! 늦어도 감 | 오늘은 패스… 내일 일찍 가자 | J / P |
| 9 | 쉬는 시간에 주로? | 다른 회원들이랑 얘기하며 회복 | 폰 보면서 혼자 쉼 | E / I |
| 10 | WOD 중 전략은? | 전체 라운드를 미리 계산하고 페이스 조절 | 일단 빠르게 시작하고 나중에 생각 | S / N |
| 11 | 코치가 폼 교정해줄 때? | 논리적으로 이해돼야 적용함 | 코치 말이면 일단 믿고 따름 | T / F |
| 12 | 크로스핏 대회 나가자는 제안에? | 일정 잡고 트레이닝 계획 즉시 세움 | "한번 생각해볼게~" 하고 미룸 | J / P |

**점수 산정 로직**:
- 각 축(E/I, S/N, T/F, J/P)별 3문항
- 다수결로 해당 축의 유형 결정
- → 최종 4글자 MBTI 유형 도출

### 4.4 MBTI 산출 로직

```typescript
// lib/calculateMBTI.ts
export function calculateMBTI(answers: Answer[]): string {
  const scores: Scores = { E:0, I:0, S:0, N:0, T:0, F:0, J:0, P:0 };

  answers.forEach((answer) => {
    scores[answer.selectedType] += 1;
  });

  const mbti =
    (scores.E >= scores.I ? 'E' : 'I') +
    (scores.S >= scores.N ? 'S' : 'N') +
    (scores.T >= scores.F ? 'T' : 'F') +
    (scores.J >= scores.P ? 'J' : 'P');

  return mbti;
}
```

URL 구조: https://crossfit-mbti.kr/result/ENFP

## 10. 바이럴 마케팅 연동 기능

- **재도전 유도**: "다른 답을 골라보면 결과가 달라질 수도? 🤔"

### 10.2 런칭 후 콘텐츠 확장
- 매주 "이번 주 가장 많이 나온 유형" 인스타 포스팅
